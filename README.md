# Hotel Booking Analysis
![hotel-jobs2-d6ebf43935](https://user-images.githubusercontent.com/67784512/211192193-3627b8ce-0082-486b-9a3b-0a61cd536c7c.jpg)

Hotel bookings depend on many factors such as type of hotels, seasonality, days of week, meals available, parking spaces, charges etc. Hence analysing the patterns available in the past data is very important to help the hotels plan well accordingly in order to benefit the business. The given data set contains booking information for a city hotel and a resort hotel, and includes information such as when the booking was made, the number of adults, children, and/or babies, and the number of available parking spaces etc, we tried to understand the customer’s’ behaviour and useful patterns in the data to help hotel managements improve the business by taking better decisions.
